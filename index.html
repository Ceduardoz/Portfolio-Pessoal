Sobre Mim

Estou trilhando meu caminho no desenvolvimento Front-End há cerca de um ano. Meu primeiro contato com esse universo veio através da minha paixão por jogos, especialmente os de puzzle, que sempre desafiaram minha lógica e criatividade.

Desde então, mergulhei nos estudos com curiosidade, sempre buscando entender como as coisas funcionam por trás das interfaces. Gosto de aprender experimentando, testando, errando e descobrindo soluções.

Atualmente, sou estudante na UNIPÊ, onde venho aprofundando meus conhecimentos e me preparando para atuar profissionalmente na área. Estou animado com as oportunidades que a tecnologia oferece e sigo focado em desenvolver interfaces que sejam funcionais, acessíveis e visualmente agradáveis.

Projetos

Timer

Um cronômetro funcional criado com HTML, CSS e JavaScript, capaz de iniciar, pausar, resetar e contar regressivamente o tempo. Projeto ideal para treinar controle de tempo, manipulação do DOM e lógica com setInterval. Interface simples e responsiva.

Calculadora

Um projeto de calculadora básica desenvolvida com HTML, CSS e JavaScript. Permite realizar operações matemáticas fundamentais como adição, subtração, multiplicação e divisão. Ideal para estudos de lógica de programação, manipulação do DOM e desenvolvimento front-end.

Lista de Tarefas

Aplicação web de lista de tarefas feita com HTML, CSS e JavaScript. Permite adicionar, marcar como concluída e remover tarefas. Projeto focado em lógica de arrays, eventos do DOM e armazenamento local com LocalStorage. Interface limpa
